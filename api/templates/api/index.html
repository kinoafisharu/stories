Описание API

Методы

/api/stories/

Получение историй.

URL: /api/stories/

Параметры

  • date_from - дата с которой получать истории.
    Пример: /api/stories/?date_from=2009-09-10
  • date_to - дата до которой получать истории.
    Пример: /api/stories/?date_to=2009-10-18 08:35:03
  • author_first_name и author_last_name - имя и фамилия автора истории.
    Пример: /api/stories/?author_first_name=Михаил&author_last_name=Иванов
